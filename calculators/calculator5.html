Calculator 5

The following are 5 calculators that are from an assignement that I worked on for my Web Design course.

This is the fifth and last calculator where we had to give each number and operator its own button along with a clear button that cleared the input field,

To give a brief explanation as to how this calculator works, all to the number and operations butons are appended into the input field. After, the user can click on the equals button to calculate what ever is in the field. If the "clear" button is clicked, The input field is cleared.

by clicking on one of the buttons, you can switch between different calculators that were used on the assignement.
